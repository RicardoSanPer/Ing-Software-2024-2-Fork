# Script 2: Caminante y Arbol Binario Ordenado

## Ejecucion

El script está ubicado en el directorio penv-1 bajo el nombre Script2.py.

Las funciones y clases se pueden ejecutar dentro del interprete de Python.

## Caminante

La funcion recibe como entrada una cadena compuesta unicamente por "u" y "d". Cadenas o valores incorrectos devuelven 
como resultado -1. La funcion cuenta solo valles que fueron recorridos completamente (por ejemplo, si el recorrido termina
a mitad de un valle, este no es contado para el total).

## Arbol Binario Ordenado

El arbol se puede inicializar pasando un valor opcional para el nodo raíz. Para la construccion del resto del arbol se puede
pasar un unico valor con la funcion insertar(), o una lista de valores con la funcion insertarLista(). El arbol no realiza
balanceos. Se asume que los valores insertados son del tipo correspondiente con el resto del arbol.
